Copyright (c) 2018 Aggelos Kolaitis

https://github.com/neoaggelos/qcmus


REQUIREMENTS
------------

* python3: PyQt5, mutagen
* cmus (duh)


INSTALLATION
------------

Just copy the source code anywhere you want and run with:

    $ python3 ./qcmus.py


FEATURES
--------

* Display current song (with album art)
* Album browser
* Library browser
* Follows system theme
* Light-weight
* Configurable
* Understands default cmus shortcuts:
        ** playback control:        c,v,z,b
        ** volume control:          +, -, {, }, [, ]
        ** seek:                    right, left, h, l, (comma), (dot)
        ** repeat and shuffle:      r, s, m, o, ctrl-r, alt-c
